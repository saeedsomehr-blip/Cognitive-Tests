Cognitive Tasks Web App (PRM & DMS-style)

A lightweight, browser-based prototype implementing PRM- and DMS-style cognitive tasks based on established protocol structure and instruction style, with customizable parameters to support flexible cognitive rehabilitation and research workflows.

Features

Two tasks: PRM-style and DMS-style

Customizable settings (timings, difficulty-related parameters, feedback, etc.)

Local data storage (no server required)

CSV export for later analysis

Bilingual UI (if enabled via settings / i18n)

How to Run

Option A: Online (GitHub Pages)

Open the GitHub Pages link for this repository:

https://saeedsomehr-blip.github.io/Cognitive-Tests/

Option B: Local

Download this repository and extract it.

Open index.html in a modern browser (Chrome/Edge/Firefox recommended).

Data & Privacy

All results are stored locally in the browser (e.g., LocalStorage).

Data can be exported as CSV from the History/Results page.

No data is uploaded to any server by default.
